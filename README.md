## Installation & Setup

1. Clone the repository:

  `git clone https://github.com/EugeniaRe/network.git`

2. Install dependencies:

  `npm install`

3. Run the application:

  `npm run dev`

### Available Scripts

  `dev` - Runs the development server

  `build` - Builds the production version

  `lint` - Runs ESLint

  `test` - Runs tests
